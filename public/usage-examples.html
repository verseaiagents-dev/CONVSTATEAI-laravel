ConvState AI - Kullanım Örnekleri

Widget'ı sitenize entegre etmek için farklı yöntemler

1. Yeni API - Önerilen Yöntem

Modern ve esnek kullanım için önerilen yöntem:

<script src="build/convstateai.min.js"></script> <script> // Widget konfigürasyonu const config = { projectId: '1', apiUrl: 'http://127.0.0.1:8000', // Development server customization: { colors: { primary: '#667eea', secondary: '#764ba2' }, position: 'bottom-right' } }; // Widget'ı yükle window.convstateai.chat.load(config).then(() => { console.log('Widget yüklendi!'); }).catch(error => { console.error('Widget yükleme hatası:', error); }); </script>

2. Eski API - Geriye Uyumluluk

Mevcut siteleriniz için geriye uyumlu yöntem:

<script src="build/convstateai.min.js"></script> <script> // Eski API kullanımı window.convstateai.load({ projectId: '1', apiUrl: 'http://127.0.0.1:8000', customization: { colors: { primary: '#667eea', secondary: '#764ba2' }, position: 'bottom-right' } }); </script>

3. Otomatik Yükleme - window.convstateaiConfig

Script yüklendikten sonra otomatik olarak widget'ı başlatır:

<script src="build/convstateai.min.js"></script> <script> // Otomatik yükleme için konfigürasyon window.convstateaiConfig = { projectId: '1', apiUrl: 'http://127.0.0.1:8000', customization: { colors: { primary: '#667eea', secondary: '#764ba2' }, position: 'bottom-right' } }; // Widget otomatik olarak yüklenecek </script>

4. Gelişmiş Kullanım - API Metodları

Widget'ı programatik olarak kontrol etmek için:

<script> // Widget'ı yükle const config = { projectId: '1', apiUrl: 'http://127.0.0.1:8000' }; window.convstateai.chat.load(config).then(() => { console.log('Widget yüklendi!'); // Widget durumunu kontrol et console.log('Widget yüklü mü?', window.convstateai.chat.isLoaded()); // Mesaj gönder window.convstateai.chat.sendMessage('Merhaba!'); // Auth bilgilerini set et window.convstateai.chat.setAuth('token123', 'user456'); // Widget'ı yok et // window.convstateai.chat.destroy(); }); </script>

5. Production Kullanımı

Production ortamında kullanım için:

<script src="https://convstateai.com/build/convstateai.min.js"></script> <script> window.convstateaiConfig = { projectId: 'YOUR_PROJECT_ID', // apiUrl belirtmezseniz otomatik olarak production URL kullanılır customization: { colors: { primary: '#667eea', secondary: '#764ba2' }, position: 'bottom-right' } }; </script>
Not: Tüm yöntemler aynı widget'ı yükler. Yeni API daha esnek ve hata yönetimi sağlar.
Başarılı! Widget artık hem eski hem yeni API'yi destekliyor ve Laravel development server ile uyumlu çalışıyor.
